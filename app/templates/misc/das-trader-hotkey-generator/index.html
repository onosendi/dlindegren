DAS Trader hotkey generator

for daniel lindegren's keyboard

If you came across this and have no idea what it is, read the article I wrote on position sizing, and how I manage my risk.

When I change my risk, it can be tedious entering new share values for every key. I wrote this generator to solve this problem, and I thought I would share it with anyone else that was interested.

This generator only generates the hotkeys for the alpha keys used to enter your positions. If you want my full hotkey file that I use for my keyboard, you can download it here.

If you are interested in the CLI version of this generator, it can be found here.

Instructions

Overview of what we will be doing:

  • Backup your current hotkey file
  • Use this generator to generate hotkeys
  • Paste the generated hotkeys into a new hotkey file
  • Load the new hotkey file into DAS Trader

Backup

Before we go messing with our hotkey file, we should back it up in case something goes wrong. Navigate to your DAS install directory, which is C:\DAS Trader Pro by default.

There are a few files in there named hotkey:

hotkey
this is an XML document
HotKey
this is a PDF document
Hotkey
this is an HTK document, the one we want

Once you have located this file:

  1. Copy the file: right click and select "copy"
  2. Paste the file: press Ctrl+V
  3. Rename the newly created file "Hotkey - Copy" to "MyHotkeys"
Note

We are going to be adding the new hotkeys to "MyHotkeys". The original "Hotkey" file you copied from will be your backup.

Generating Hotkeys

Use the form on this page to generate your hotkeys, then copy the output to your clipboard.

Edit Your Hotkey File

Now we are going to paste the hotkeys we generated into your newly created "MyHotkeys" file. To do this, we need to open the file up in an editor:

  • Right click "Myhotkeys" and select "Open with...", then select "Notepad"
  • Paste the lines that you generated at the bottom of the file, then save it

Load Your Hotkeys Into DAS

Now it is time to load your new hotkey file into DAS Trader:

  1. Open DAS
  2. From the menu bar, open your hotkey configuration by selecting "Setup", then "Hotkey"
  3. Click "Browse" at the bottom of the window, then open your newly created hotkey file "MyHotkeys"
  4. Click "Ok", and you are finished!
